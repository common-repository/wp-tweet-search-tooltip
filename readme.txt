== Description ==

Adds a tooltip on an chosen keyword for a search via twitter.
Simply choose the keyword by surrounding with the shorttag.
<code>[tweetsearch]Keyword[/tweetsearch]</code>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
